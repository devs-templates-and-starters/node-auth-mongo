# node-auth-mongo
____
Node(Express), MongoDB based app with Authentication Boilerplate.\
It can be used to get started with any project to have JWT authentication.


## How to install

1. clone the repo
2. run `npm install`
3. create `.env` file as given in `.env.example`
4. run `npm run dev`
